## Included Fixes

- Event-copy links use opaque cross-device keys instead of source-device Room IDs.
- Room migration 4 to 5 preserves existing events and transactions while assigning event keys.
- Receipt OCR prefers large payment amounts over available-balance values, including when OCR misses the currency symbol.
- Receipt review is compact and human-readable; raw JSON and confidence percentages are not shown.
- Unlabelled or edited amounts require explicit confirmation against the original receipt.
- Receipt evidence and the Room write run through one awaited operation with cleanup on database failure.
- OCR progress uses a neutral on-device indicator without terminal-like scripts or fake delays.
- Transaction cards do not expose another uploader's local email username.
- Privacy and Play Data safety documentation includes ML Kit diagnostics and usage metadata.

## Important Limits

- This is a beta for testing, not the only record for important or irreplaceable money data.
- Each installation owns an independent local ledger. There is no account, authentication, authorization, or cross-device synchronization.
- Event-copy links do not add live members or synchronize balances, transactions, receipt evidence, or custom fields.
- Public/private is a local marker, not access control.
- OCR can be wrong. Review every amount and detail against the original receipt.
- There is no export, restore, cloud backup, or recovery after uninstall/device loss.
- The app records ledger evidence; it does not collect, hold, transfer, or invest money.
- Complete physical-device breadth, qualified legal review, and market-demand evidence remain incomplete.

## Installation

Download only from the official GitHub Release page. Verify the SHA-256 above before installation. Android may ask for per-source permission; Developer options are not required for normal APK installation.

Do not uninstall beta.1 to upgrade because uninstalling deletes local data. Install beta.2 over beta.1 so Android can preserve app data and run the explicit Room migration.
