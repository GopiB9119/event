# Writing System

## Core Rule

Never waste a sentence.

## Standards

- Every sentence should add information.
- Prefer concrete nouns and verbs.
- Avoid inflated adjectives.
- Use evidence when making claims.
- Avoid AI cliches.
- Write for the reader's next action.
- Keep powerful language tied to real implementation.

## Modes

- Executive: decision, impact, risk.
- Engineer: system, constraints, tradeoffs.
- User: action, result, recovery.
- Security: threat, control, evidence, gap.
