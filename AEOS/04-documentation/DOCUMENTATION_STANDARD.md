# Documentation Standard

Documentation is a product surface for future engineers and AI agents.

## Required Qualities

- Current
- Specific
- Linked
- Decision-oriented
- Searchable
- Honest about limits

## Common Document Types

- README
- Architecture overview
- ADR
- Feature spec
- Security note
- Runbook
- Migration guide
- Test plan
- Postmortem
- Changelog
- Release notes

## Document Structure

```text
Summary
Context
Current behavior
Desired behavior
Constraints
Decision
Validation
Risks
References
```
