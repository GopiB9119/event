# Glossary

- **AEOS**: AI Engineering Operating System.
- **Invariant**: A condition that must remain true for correctness.
- **Trust boundary**: Where untrusted input enters a system.
- **Evidence gate**: A condition that blocks a mutation or claim until required evidence exists.
- **Confidence gate**: A threshold that blocks low-quality extraction from mutating state.
- **Duplicate guard**: Logic that prevents repeated input from creating repeated effects.
- **Durable identifier**: A persisted identity used across state transitions and relationships.
- **Evidence record**: Structured, reviewable data supporting a decision, mutation, or verification claim.
