# Glossary

- **AEOS**: AI Engineering Operating System.
- **Invariant**: A condition that must remain true for correctness.
- **Trust boundary**: Where untrusted input enters a system.
- **Receipt integrity**: The property that ledger entries are backed by validated receipt evidence and visible confidence signals.
- **Confidence gate**: A threshold that blocks low-quality extraction from mutating state.
- **Duplicate guard**: Logic that prevents repeated input from creating repeated financial effects.
- **Member ID**: Persisted identifier for a person in an event.
- **Proof record**: Structured JSON stored as evidence for a transaction.
