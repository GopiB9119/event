# Community Code Of Conduct

## Our Standard

Community Ledger project spaces must remain respectful, evidence-focused, and safe for people discussing sensitive financial-record software.

Expected behavior includes:

- discussing people and technical decisions respectfully
- challenging claims with evidence rather than attacking contributors
- protecting personal, receipt, payment, account, and security data
- disclosing conflicts of interest and uncertainty
- accepting correction when evidence changes a decision

Unacceptable behavior includes:

- harassment, discrimination, threats, or unwanted sexual attention
- publishing another person's private information
- posting receipt images, contact details, payment identifiers, credentials, or verification codes
- impersonating organizers, maintainers, or other contributors
- spam, scams, deceptive fundraising, investment solicitation, or payment requests
- knowingly submitting fabricated evidence, false security claims, or malicious code

## Scope

This code applies to Issues, Pull Requests, Discussions, project boards, release communication, and other spaces represented as part of Community Ledger.

## Reporting And Enforcement

Report conduct concerns privately to **banothgopikrishna19@gmail.com**. Do not expose sensitive evidence in a public issue.

The publisher may edit or remove content, close or lock conversations, reject contributions, or restrict participation when necessary to protect users and the project. Decisions should be proportionate, documented privately when sensitive, and revisited if new evidence changes the facts.

No response-time guarantee is offered by this independent project.
