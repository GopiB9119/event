Product truth

A ledger, not a money service

Money stays outside

The app does not collect donations, move funds, hold money, or offer investments. It records information an organizer reviews.

OCR stays on device

Google ML Kit reads selected receipt images locally. OCR can be wrong, so weak or duplicate evidence is blocked from totals.

No hidden tracking

No ads, analytics SDK, account service, or synchronized transaction server. Update checks run only when requested.

The working loop

Every number should have a reason

Community Ledger is designed for organizers who need a simple local record of money in, money out, and the evidence behind each entry.

  1. Create a local event.Name the event and choose a local visibility marker.
  2. Record people and entries.Link contributions and expenses to members or vendors.
  3. Review receipt evidence.Check the amount, reference, receiver, confidence, and warnings.
  4. Save only what is supported.Positive amounts and duplicate checks protect the ledger totals.

Important limitation

Event links copy. They do not sync.

An event-copy link adds an independent event shell on another phone. It does not create a shared account, verify the organizer, enforce private access, or synchronize members, receipts, transactions, and balances.

Before you install

  • Never send money only because an event link looks legitimate.
